13-04-2026 - the ipod saga

i've owned my ipod for around a year, having gotten it second-hand in march of 2025. in that time, the entire ipod has corrupted itself three times, experienced two microsd card swaps, and has seen two rockbox installs. i've barely been able to use the thing for more than like two or three weeks total in that entire timespan.

i think (keyword think, i literally cannot know for sure) this was originally caused by a faulty rockbox install when i first got the thing, but it certainly does not explain the frequent self-corruption. perhaps it's a case of simple user error but i really really really doubt that; even reddit said it was a problem with the data cable, so this cannot be an uncommon occurance.

however, as of tonight, i can report that i have a brand new install of rockbox on a now fully functioning ipod, with 50gb of flac files playing smoothly and beautifully as if im playing off my computer itself. yippi

it took a lot of trial and error and a lot of steps; for one, having recently swapped to linux and rockbox only being compatible with windowpods (that is, ipods with firmware installed from a windows installation of itunes because the filesystems are different or something), i had to set up a windows 11 virtual machine with qemu. suffice to say that was a complete pain in the ass considering windows 11 itself performs sluggishly on bare metal, much less in a virtual environment, and i had to tweak with the os/debloat for about an hour after install.

post-install was also a bit of a pain. apple's shitty ass windows store app meant specifically for detecting apple devices did not install the ipod software framework thing that i completely forgot the actual name of. that required a bit of manual tweaking and generally watching loading bars load very very slowly.

but, i will say, after getting the ipod swapped over to a windowpod, it was a breeze. there were no hiccups in installing rockbox and transferring my music over. :DDDDD

i love my ipod !
